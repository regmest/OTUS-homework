Идея проекта

MySkills

Сайт, где можно следить за своим прогрессом по получению навыков

Основное:
    - Страница пользователя с информацией
    - Страница навыков пользователя, где можно добавить новый или отредактировать существующий
    - Общая страница всех навыков всех пользователей, откуда пользователь может добавить какой-либо скилл себе
    - Скилл имеет статус (not started/in progress/finished/failed)
    - Скиллы имеют теги категорий (пример: "Научиться рисовать" имеет теги "Творчество" и "Искусство")
    - На странице скилла пользователь может добавлять комментарии о своих успехах
    - Пользователь может добавлять друзей

Возможно:
   - Скиллы могут иметь родительские и дочерние скиллы (пример: скилл "Научиться рисовать" имеет подскилл "Научиться рисовать акварелью")
   - Пользователь может получать медальки за какие-либо достижения
