Идея проекта

MySkills

Сайт, где каждый может следить за своим прогрессом по получению навыков

Основное:
    - На странице профиля будут размещены скиллы, над которыми человек работает
    - Скилл имеет бар заполнения и статус (not started/in progress/finished/failed)
    - Скиллы могут иметь родительские и дочерние скиллы (пример: скилл "Научиться рисовать" имеет подскилл "Научиться рисовать акварелью")
    - Скиллы имеют теги направлений (пример: "Научиться рисовать" имеет теги "Творчество" и "Искусство")
    - Пользователь может получать ачивки (медальки)
    - На странице скилла пользователь может добавлять комментарии о своих успехах
    - Пользователь может добавлять друзей
    - Можно оставлять комментарии на страницах пользователей


Реализация:
        1. При авторизации пользователь попадает на свою страницу
        2. Страница пользователя:
            - фото (emoji)
            - скиллы пользователя
        3. Страница скилла конкретного пользователя
            - картинка (emoji)
            - бар заполнения
            - комментарии
            - прогресс рассчитывается так:
                    пользователь задает количество тем, которые нужно пройти
                    каждую пройденную тему пользователь отмечает
                    progress_percent = completed / all
